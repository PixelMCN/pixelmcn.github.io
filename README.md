## Installation & Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Customization

All content is easily customizable in `src/App.tsx`:
- Update social links
- Modify project data
- Adjust colors in Tailwind config
- Customize animations in GSAP sections
