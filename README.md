# Minecraft Portfolio

Welcome to my Minecraft-themed personal website! This site showcases my projects and provides information about me.

## Overview

This website is designed to reflect the aesthetics of Minecraft, featuring blocky designs and earthy colors. It serves as a portfolio to display my work and connect with visitors.

## Project Structure

- `src/css/style.css`: Contains styles for the website.
- `src/js/main.js`: Includes JavaScript for interactive elements.
- `src/pages/about.html`: About page with information about me.
- `src/pages/contact.html`: Contact page with a form for inquiries.
- `src/pages/projects.html`: Projects page showcasing my work.
- `index.html`: Main entry point linking to all pages.
- `package.json`: Configuration file for npm.

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/minecraft-portfolio.git
   ```
2. Navigate to the project directory:
   ```
   cd minecraft-portfolio
   ```
3. Install dependencies (if any):
   ```
   npm install
   ```
4. Open `index.html` in your browser to view the website.
